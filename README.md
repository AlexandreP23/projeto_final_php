# 🧠 MedLang AI

**MedLang AI** é uma plataforma web em desenvolvimento que utiliza **inteligência artificial** para traduzir laudos médicos com foco na **adequação de termos técnicos** para diversos idiomas,
facilitando a comunicação entre profissionais da saúde e pacientes em contextos multilíngues.

## 🚀 Sobre o Projeto

O projeto foi iniciado como uma iniciativa da nossa startup, em parceria com **Lucas Alves** e **Isadora de Abreu**. A primeira versão do sistema inclui um **CRUD** para cadastro de usuários 
e laudos, além de uma interface inicial para acesso às traduções dos laudos geradas por IA.

---

## 🛠️ Tecnologias Utilizadas

- **PHP** – Backend da aplicação
- **MySQL** – Banco de dados relacional
- **XAMPP** – Ambiente local de desenvolvimento (Apache + MySQL)
- **HTML/CSS** – Estrutura e estilização das páginas
- **JavaScript** – Funcionalidades interativas (em partes específicas)

---

## 📂 Funcionalidades

- Cadastro de usuários
- Login e autenticação
- Upload de laudos médicos
- Visualização dos laudos traduzidos automaticamente
- Tradução técnica com base em IA (em desenvolvimento)
